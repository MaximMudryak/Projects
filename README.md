# Writeups

A collection of writeups, notes, and methodologies from CTF challenges and security labs (TryHackMe, HackTheBox, and others).

## Contents

- **Anonymous_Writeup.pdf** – TryHackMe (Linux, Medium): FTP anonymous login, SMB share enumeration, reverse shell via cron-executed script, SUID `/usr/bin/env` privilege escalation.
- **Attacktive_Directory_Writeup.pdf** – TryHackMe (Windows AD, Medium): Kerberos user enumeration with Kerbrute, AS-REP roasting, SMB share access, DCSync via secretsdump, Pass-the-Hash with Evil-WinRM.

## Disclaimer

These writeups are for educational purposes only. All targets are legal CTF/lab environments.
